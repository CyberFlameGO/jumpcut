Jumpcut 0.71 contains a number of small bug fixes and minor improvements. No new functions or UI changes are included in this release. Notable improvements:

  • The Sparkle auto-updater code was previously not checking on launch, even if the preference was active
  • The hotkey recorder now displays correctly in Dark Mode
  • Jumpcut 0.71 now supports El Capitan and Sierra
  • The bezel size and positioning now resets when the display size changes, as from unplugging a monitor
  • Very large clips—thousands of lines or more—now render more smoothly in the bezel
  • The minimum number of clips to remember has been raised from 1 to 10
