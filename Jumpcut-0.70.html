This is the first release of Jumpcut in 10 years, and includes a number of internal changes that will be required to run Jumpcut under versions of macOS Catalina (10.15). Users running versions prior to High Sierra (10.13) should continue to use Jumpcut 0.63. If you're on an old version of Jumpcut that can't update automatically, please visit the new GitHub page, https://snark.github.io/jumpcut/ and download it there.

  • 64-bit compatible; required for future versions of macOS
  • Updated to reflect modern macOS security practices, including Accessability entitlements, code-hardening, and notarization
  • Improved support for paste behavior on non-American keyboards
  • Restored “Clear All” menu item
  • Retina-ready “Jumpcut logo” menu bar icon (now supporting dark mode!)
  • Known transient sources of content (such as TypeIt4Me and TextExpander) are now ignored
  • Option to ignore known sources of sensitive data, such as KeePassXC or 1Password

Note that development, including bug reports, has been moved to GitHub. Thanks for using Jumpcut, for however many years you've been using it!
