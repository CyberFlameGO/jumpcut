Jumpcut 0.75 improves an edge case where certain transient content, like that from many password managers, was incorrectly being recorded.

  • Altered the check for transient types (and, optionally, concealed types) to use a less brittle method that does not require actually loading the pasteboard item. (Thanks to GitHub users acca and maybeageek for the bug reports.)
  • Minimum macOS/OS X version has been set back to 10.10 from 10.11.
  • The "Clear All" menu item's warning may now be disabled. (Set the "askBeforeClearingClippings" preference back to true to re-enable it if needed.)
